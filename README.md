# ESCANEAR RUTER v3.0

Scanner de red para Termux  en Android.

escanearuter | Version para Ubuntu dentro de proot-distro |
| escanearuter-termux | Version para Termux puro sin Ubuntu |

## Instalacion

git clone https://github.com/cccrrhhl-create/escanearuter-.git
cd escanearuter-
bash install.sh

El instalador te preguntara que version instalar
y descargara las dependencias correctas.
